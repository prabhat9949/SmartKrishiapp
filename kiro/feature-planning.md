## Core Objective
Build a smart farming assistant that helps farmers take better decisions using AI and data.

## Feature Breakdown

### 1. Crop Disease Detection
- Farmer uploads or captures an image of crop leaf
- AI model analyzes the image
- Disease name and confidence score shown
- Basic treatment suggestions provided

### 2. Weather-Based Advisory
- Fetch real-time weather data
- Show temperature, humidity, rainfall
- Advisory messages based on weather conditions

### 3. Krishi Mitra (AI Chat Assistant)
- Conversational interface for farmers
- Simple language responses
- Helps with crop-related questions

### 4. Farmer Dashboard
- Central hub for all features
- Shows weather, alerts, recent activity
- Clean and minimal UI

### 5. Crop Knowledge Section
- Best practices for major crops
- Seasonal guidance
- Fertilizer and irrigation tips

## Feature Prioritization (Hackathon Scope)
High Priority:
- Disease Detection
- Dashboard
- Weather Advisory

Medium Priority:
- AI Chat
- Crop Knowledge

Low Priority:
- Advanced analytics
- Marketplace integration
